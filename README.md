# vehicle-number-identification

`mlmodel.py` contains the OpenCV model.

`requirements.txt` contains the necessary Python libraries for the project.
To setup python packages, run:
```
pip install -r requirements.txt
```

**In the terminal, run:**

```
cd backend
python app.py
```
